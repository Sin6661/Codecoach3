Lesson 1: Get Your Google Places API Key
Step 1: Go to Google Cloud Console

Open this link in a new tab:

→ Go to Google Cloud Console

What you'll see: A login page

What to do: Log in with your Google account (create one if you don't have it)

Step 2: Create a New Project
  1. Look at the top left - you'll see "Google Cloud" logo
  2. Right next to it, there's a dropdown that says "Select a Project"
  3. Click it
  4. A popup appears - click "NEW PROJECT" (top right)
  5. Enter any name (example: "AssistaSale")
  6. Click "CREATE"
  7. Wait 30 seconds for it to load
Step 3: Enable Places API
  1. In the left sidebar, click "APIs & Services"
  2. Click "Library"
  3. Search for "Places API"
  4. Click on "Places API"
  5. Click "ENABLE"
  6. Wait for it to enable (should say "API enabled")
Step 4: Create an API Key
  1. In the left sidebar, click "Credentials"
  2. Click "CREATE CREDENTIALS" (top button)
  3. Select "API Key"
  4. A popup shows your new API key
  5. COPY THIS KEY - you'll need it
⚠️ Keep this key SECRET. Don't share it publicly!
✓ You're Done with Step 1!

You now have a Google Places API key.

Save it somewhere safe (notepad, etc.)

Lesson 2: Get Your Groq API Key
Step 1: Go to Groq Console

Open this link:

→ Go to Groq Console

What you'll see: A signup page

Step 2: Create a Free Account
  1. Click "Sign Up" (or "Get Started Free")
  2. Enter your email
  3. Create a password
  4. Check your email for a verification link
  5. Click the link to verify
  6. You're in!
✓ Groq is completely FREE - no credit card needed!
Step 3: Get Your API Key
  1. You should already be in the console at console.groq.com/keys
  2. If not, go back to the link above
  3. Look for a button "CREATE API KEY"
  4. Click it
  5. A new key appears
  6. COPY THE ENTIRE KEY
Step 4: Save Your Key

You now have a Groq API key. Save it in the same place as your Google key.

Format: It looks like a long random string starting with "gsk_"
